# Skin Disease Classification Using Deep Learning

## Overview
This project uses deep learning models to classify multiple skin diseases using dermatoscopic images.

The project compares the performance of:
- ResNet50
- EfficientNet-B0
- EfficientNet-B3
- DenseNet121

and includes advanced evaluation techniques such as Grad-CAM visualization, ROC curves, confusion matrices, and precision-recall curves.

---

## Features
- Transfer Learning using pretrained CNN models
- Image augmentation
- Weighted loss function for class imbalance handling
- Early stopping
- ROC Curve analysis
- Precision-Recall Curves
- Confusion Matrix visualization
- Grad-CAM explainability
